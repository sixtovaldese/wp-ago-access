# aGo Access

Accessibility toolbar and automatic fixes for WordPress. WCAG 2.2 oriented. **Free, no Pro tier.**

[English](#english) · [Español](#espanol)

---

## English

aGo Access adds a floating accessibility toolbar with 11 tools and 7 automatic background fixes to your WordPress site.

### Features

- Floating toolbar with 11 user tools: text size, spacing, contrast modes, dyslexia font, reading guide, read aloud, grayscale, stop animations, and more.
- 7 background fixes: ARIA landmarks, lang attribute, skip-to-content, focus visible, viewport meta, tabindex normalization, alt text warnings.
- Accessibility score scanner for your homepage.
- Lightweight (< 10KB on frontend).
- Preferences saved per visitor.
- No external services.
- English, Spanish (es_ES) and Brazilian Portuguese (pt_BR) bundled.

### Installation

1. Upload the `ago-access` folder to `/wp-content/plugins/`.
2. Activate the plugin.
3. Visit any page on the frontend: the toolbar button appears in the configured corner.

## Español

aGo Access agrega una barra flotante de accesibilidad con 11 herramientas y 7 correcciones automaticas en segundo plano.

### Caracteristicas

- Barra flotante: tamano de texto, espaciado, contraste, fuente para dislexia, guia de lectura, lectura en voz alta, escala de grises, parar animaciones.
- 7 correcciones automaticas: landmarks ARIA, atributo lang, skip-to-content, foco visible, viewport, tabindex, avisos de alt text.
- Scanner de score de accesibilidad para la home.
- Liviano (< 10KB en frontend).
- Preferencias guardadas por visitante.
- Sin servicios externos.
- Ingles, espanol (es_ES) y portugues brasileno (pt_BR) incluidos.

### Instalacion

1. Sube la carpeta `ago-access` a `/wp-content/plugins/`.
2. Activa el plugin.
3. Visita cualquier pagina del frontend, la barra aparece en la esquina configurada.
